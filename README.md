# 🚀 Projeto Web Simples

Este projeto consiste no desenvolvimento de uma aplicação web básica utilizando HTML e CSS, com versionamento via Git, hospedagem no GitHub e deploy na Vercel.

---

## 📌 Sobre o Projeto

O objetivo deste projeto é aplicar conceitos fundamentais de desenvolvimento web, incluindo:

* Estruturação de páginas com HTML
* Estilização com CSS
* Versionamento de código com Git
* Publicação de repositório no GitHub
* Deploy de aplicação na Vercel

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Git
* GitHub
* Vercel

---

## 📂 Estrutura do Projeto

```
projeto-web-simples/
│
├── index.html
├── style.css
└── README.md
```

---

## 🎨 Funcionalidades

* Layout responsivo simples
* Menu de navegação
* Seções: Início, Sobre e Contato
* Estilização moderna com CSS
* Botão interativo com JavaScript básico

---

## 🔄 Versionamento

O projeto utiliza versionamento com Git, com marcação de versões estáveis através de tags:

* 1.0.0 → Versão inicial
* 1.1.0 → Melhorias no layout e funcionalidades

---

## 🌐 Deploy

O projeto foi publicado utilizando a plataforma Vercel, permitindo acesso online à aplicação.
